### Installation

# Clone the repository
git clone https://github.com/Jyothi-CY/Onboarding_Dashboard.git
cd Onboarding_Dashboard

# Install dependencies
```bash
npm install
```
# Run the app
```bash
npm run dev
```

The app will start on http://localhost:5173 or the next available port.
